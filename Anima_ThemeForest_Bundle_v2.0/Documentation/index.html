Introduction

Responsive Pet Rescue and Animal Shelter Bootstrap 3 Template

Anima is Creative, Ultra Responsive & Unique design Animal Shelter & Pet Rescue Template. It specially designed for Pet Rescue & Animal Shelter Websites. It can be also used for Non-Profit Charity Website, Charity organization, Event Websites, or any donation & event based websites.

Anima is Responsive Bootstrap 3 Pet Rescue and Shelter Template built with Bootstrap 3.3.7, HTML5 and CSS3. Well Organized and valid Code.

Features

  • HTML5 and CSS3
  • Twitter Bootstrap 3.3.7
  • Fully Responsive Design
  • LESS CSS
  • Parallax Effect
  • MailChimp Integration
  • Working Contact Form
  • Shortcode, Blog, Single and Donation Pages included.
  • Google Fonts: Oswald and Raleway
  • 360 Font Icons (Font Awesome)
  • Clean & Well Commented Code
  • Well Documented
  • Cross Browser Compatible - IE9+, Chrome, Firefox, Safari, Opera

JavaScript and Ajax

Anima imports following Javascript files.

  1. jQuery
  2. Custom scripts
  3. Google Map Api
  4. MixitUp
  5. Nivo LightBox
  6. etc

All code are beautifully written. You can customize easily. Just look the js code.
Example: If you want to Customize NewsLetter Subscription Url, just find the following block into the custom.js and change what you need:
 

/*

|----------------------------------------------------------------------------

| AJAX SUBSCRIBE

|----------------------------------------------------------------------------

*/

$("#subscribe").ajaxChimp({

    callback: mailchimpCallback,

    url: "http://codepassenger.us10.list-manage.com/subscribe/post?u=6b2e008d85f125cf2eb2b40e9&id=6083876991" 
	// Replace your mailchimp post url inside double quote "".  

});
